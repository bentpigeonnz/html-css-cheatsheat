HTML Basics
Element Description Example
<!DOCTYPE html> Defines the document type and version of HTML <!DOCTYPE html>
<html> Root element of an HTML page <html>...</html>
<head> Container for metadata (data about data) <head>...</head>
<title> Sets the title of the document <title>Page Title</title>
<body> Contains the content of the document <body>...</body>
<p> Defines a paragraph <p>This is a paragraph.</p>
<a> Defines a hyperlink <a href="https://example.com">Link</a>
<img> Embeds an image <img src="image.jpg" alt="Description">
<div> Defines a division or section <div>Content here</div>
<span> Defines a section in a document <span>Inline text</span>
<ul> Defines an unordered list <ul><li>Item 1</li><li>Item 2</li></ul>
<ol> Defines an ordered list <ol><li>Item 1</li><li>Item 2</li></ol>
<table> Defines a table <table>...</table>
<tr> Defines a row in a table <tr>...</tr>
<th> Defines a header cell in a table <th>Header</th>
<td> Defines a standard cell in a table <td>Data</td>
<form> Defines an HTML form for user input <form>...</form>
<input> Defines an input control <input type="text" name="name">
<button> Defines a clickable button <button>Click me</button>
<label> Defines a label for an input element <label for="inputId">Label Text</label>
<select> Defines a dropdown list <select><option>Option 1</option></select>
<option> Defines an option in a dropdown list <option value="value">Option Text</option>
<textarea> Defines a multiline text input control <textarea rows="4" cols="50">Text here</textarea>
<link> Defines the relationship between a document and an external resource (most used to link stylesheets) <link rel="stylesheet" href="styles.css">
<script> Defines a client-side script <script src="script.js"></script>
<meta> Defines metadata about an HTML document <meta charset="UTF-8">
<header> Defines a header for a document or section <header>...</header>
<footer> Defines a footer for a document or section <footer>...</footer>
<article> Defines independent, self-contained content <article>...</article>
<section> Defines a section in a document <section>...</section>
<aside> Defines content aside from the content it is placed in (like a sidebar) <aside>...</aside>
<nav> Defines navigation links <nav>...</nav>
<main> Defines the main content of a document <main>...</main>
<figure> Defines self-contained content, like illustrations, diagrams, photos, code listings, etc. <figure>...</figure>
<figcaption> Defines a caption for a <figure> element <figcaption>Caption text</figcaption>
<blockquote> Defines a section that is quoted from another source <blockquote>...</blockquote>
<code> Defines a piece of computer code <code>console.log('Hello, World!')</code>
<pre> Defines preformatted text <pre>Preformatted text here</pre>
<strong> Defines important text <strong>Important text</strong>
<em> Defines emphasized text <em>Emphasized text</em>
<small> Defines smaller text <small>Small text</small>
<mark> Defines marked or highlighted text <mark>Highlighted text</mark>
<sub> Defines subscript text <sub>Subscript</sub>
<sup> Defines superscript text <sup>Superscript</sup>
<time> Defines a specific time (or range of time) <time datetime="2023-10-01">October 1, 2023</time>
<progress> Defines the progress of a task <progress value="50" max="100">50%</progress>
<meter> Defines a scalar measurement within a known range <meter value="0.6" min="0" max="1">60%</meter>
<details> Defines additional details that the user can view or hide <details><summary>More info</summary>Details here</details>
<summary> Defines a summary for a <details> element <summary>Summary text</summary>
<dialog> Defines a dialog box or other interactive component <dialog>Dialog content</dialog>
CSS Basics
Property Description Example
color Sets the color of text color: red;
background-color Sets the background color of an element background-color: blue;
font-size Sets the size of the font font-size: 16px;
font-family Sets the font family for text font-family: 'Ubuntu', sans-serif;
font-weight Sets the weight (boldness) of the font font-weight: bold;
text-align Sets the horizontal alignment of text text-align: center;
line-height Sets the height of a line box line-height: 1.5;
margin Sets the margin area on all four sides of an element margin: 10px;
padding Sets the padding area on all four sides of an element padding: 10px;
border Sets the border around an element border: 1px solid black;
width Sets the width of an element width: 100%;
height Sets the height of an element height: 200px;
display Sets the display behavior of an element display: block;
position Sets the type of positioning method used for an element position: relative;
top Sets the top position of an element top: 10px;
left Sets the left position of an element left: 20px;
z-index Sets the stack order of an element z-index: 100;
overflow Sets what happens when content overflows an element's box overflow: hidden;
float Sets an element to float to the left or right float: left;
clear Sets whether an element can be next to floating elements clear: both;
flex Sets the flexible length of a flex item flex: 1;
CSS Selectors
Selector Description Example
* Selects all elements * { color: black; }
element Selects all elements of a specific type p { font-size: 16px; }
.class Selects all elements with a specific class .my-class { color: red; }
#id Selects an element with a specific ID #my-id { background-color: blue; }
element.class Selects elements of a specific type with a specific class div.my-class { margin: 10px; }
element#id Selects a specific element with a specific ID h1#main-title { text-align: center; }
element, element2 Selects multiple elements h1, h2 { font-weight: bold; }
element element2 Selects element2 that is a descendant of element div p { color: green; }
element > element2 Selects element2 that is a direct child of element ul > li { list-style-type: none; }
element + element2 Selects element2 that is immediately preceded by element h1 + p { margin-top: 0; }
element ~ element2 Selects all element2 that are preceded by element h1 ~ p { color: gray; }
:hover Selects an element when it is hovered over a:hover { text-decoration: underline; }
:focus Selects an element when it is focused (e.g., input fields) input:focus { border-color: blue; }
:first-child Selects the first child of an element li:first-child { font-weight: bold; }
:last-child Selects the last child of an element li:last-child { font-style: italic; }
:nth-child(n) Selects the nth child of an element li:nth-child(2) { color: red; }
:not(selector) Selects elements that do not match the specified selector p:not(.special) { color: gray; }
:checked Selects checked input elements (e.g., checkboxes, radio buttons) input:checked { background-color: yellow; }
:disabled Selects disabled input elements input:disabled { opacity: 0.5; }
:enabled Selects enabled input elements input:enabled { cursor: pointer; }
:active Selects an element that is being activated by the user a:active { color: orange; }
:visited Selects links that have been visited a:visited { color: purple; }
:link Selects links that have not been visited a:link { color: blue; }
:before Inserts content before an element p:before { content: "Note: "; font-weight: bold; }
:after Inserts content after an element p:after { content: " (end)"; }
Common CSS Properties
Property Description Example
background-image Sets a background image for an element background-image: url('image.jpg');
background-size Sets the size of the background image background-size: cover;
border-radius Sets the radius of the element's corners border-radius: 5px;
box-shadow Adds shadow effects around an element's frame box-shadow: 2px 2px 5px gray;
opacity Sets the opacity level for an element opacity: 0.8;
transition Defines the transition effect when a property changes transition: all 0.3s ease;
